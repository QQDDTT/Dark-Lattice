# 个人形象照拍摄与视觉规范 (Photo Design)

## 1. 核心视觉目标
打造具有“研究者”气息的专业形象，确保照片风格与 **Dark Lattice** 网站配色、调性高度统一，实现背景的“无缝呼吸”感。

---

## 2. 布光方案 (Lighting Setup)

为了实现极客感的高对比度视觉，建议采用以下布光逻辑：

- **主灯 (Key Light)**：侧前方 45°，使用柔光箱，保留面部轮廓阴影（Rembrandt Lighting）。
- **轮廓灯 (Rim Light)**：侧后方，颜色使用 **科技蓝 (#3B82F6)** 滤色片，勾勒鬓角与肩膀，表现科技感。
- **环境光 (Ambient)**：全黑或极低亮度，确保背景为纯色或深灰色，便于后期透明化处理。

---

## 3. 拍摄细节 (Shooting Details)

### 3.1 着装要求
- **首选**：深灰色高领针织衫、黑色工装/西装、深蓝衬衫。
- **质感**：避免亮面反光材质，选择哑光、有纹理的织物，能更好地吸收光线。

### 3.2 姿势与表情
- **工作态**：手持平板电脑、侧身操作键盘或低头阅读，展现专注感。
- **冷静态**：双手交叉，眼神直视镜头，展现作为研究者的自信。
- **微表情**：不建议露齿大笑，以平缓、深邃的表情为主。

---

## 4. 后期校色 (Post-Processing)

| 参数项 | 建议值 | 理由 |
| :--- | :--- | :--- |
| **对比度 (Contrast)** | `+15 ~ +25` | 增强视觉冲击力 |
| **色温 (Temp)** | `-10 ~ -15 (偏冷)` | 符合科技冷调风格 |
| **暗角 (Vignette)** | `中度` | 聚焦面部，使边缘融入深色网页背景 |
| **HSL - 蓝色** | 调整至 `#3B82F6` | 与主色调匹配 |

---

## 5. 输出规范
- **格式**：WebP (含 Alpha 通道)。
- **尺寸**：1024px x 1024px。
- **命名**：`avatar-cool-blue.webp`。
