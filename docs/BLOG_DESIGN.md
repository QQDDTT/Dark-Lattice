# 个人博客设计文档 (Blog Design)

## 1. 核心定位
以文章内容为核心，提供沉浸式的学术与技术阅读体验。支持复杂的代码展示、数学公式及学术引用。

---

## 2. 页面模块规划

| 模块 | 核心功能 |
| :--- | :--- |
| **Index（列表页）** | 时间轴展示、分类/标签云、搜索过滤 |
| **Content（正文页）** | 目录导航、阅读进度条、学术引用脚注 |
| **Research（研究专栏）** | 论文摘要卡片、BibTeX 导出、数据图表嵌入 |

---

## 3. 功能细节深化

### 3.1 代码高亮方案
- **主题**：采用 `Nightfield` 风格定制的高对比度暗调主题（类似 One Dark Pro 或 Monokai Pro）。
- **交互**：一键复制功能，显示代码语言标签。

### 3.2 学术支持 (Academic Support)
- **公式**：集成 **KaTeX**，实现毫秒级公式渲染，支持 LaTeX 语法。
- **引用**：
  - 点击参考文献序号（如 `[1]`）平滑滚动至页面底部。
  - 支持弹出式预览（Hover Tooltip）论文标题。
  - 提供 `BibTeX` 格式的复制入口。

### 3.3 国际化适配 (Multi-language Support)
- **翻译准则**：遵循 [国际化方案设计](./I18N_DESIGN.md) 中的特定内容规范。
- **学术优先**：对于 BibTeX 和复杂公式，默认优先保持英文版的一致性。

### 3.4 响应式体验
- **移动端**：正文左右边距自适应，目录折叠至侧边栏或底部抽屉。
- **排版**：中英文混排优化（使用 `pangu.js` 思路进行自动间距调整或手动控制）。

---

## 4. 交互与功能细节补全

### 4.1 评论系统 (Giscus)
- **方案**：采用基于 GitHub Discussions 的 **Giscus**。
- **自定义样式**：需基于 Nightfield 调色盘定制 CSS，确保评论区与整体暗色调无缝融合。
- **加载逻辑**：仅在用户滚动到页面底部或点击“展开评论”时动态加载。

### 4.2 全文检索与 3D 交互联动 (Fuse.js & 3D Interaction)
- **方案**：采用轻量级客户端搜索库 **Fuse.js**。
- **配置**：构建时生成 `search.json` 索引。
- **交互**：
  - 全局快捷键 `Cmd+K` 或 `Ctrl+K` 唤起。
  - **3D 反馈**：搜索输入时，背景场景产生实时波纹（Ripple）或粒子汇聚效果。
  - **过滤动效**：列表项的过滤不再是简单的隐藏，而是通过 3D 轴向旋转移出视野。

### 4.3 3D 渲染引擎 (Rendering Engine)
- **技术选型**：引入 **Three.js** 或 **Spline Runtime** 作为 3D 核心渲染器。
- **性能优化**：
  - 采用 **Instanced Mesh** 处理大量的格阵粒子。
  - **LOD (Level of Detail)**：根据滚动位置及元素可见性动态降低/提升渲染复杂度。
  - **OffscreenCanvas**：在 Web Worker 中处理渲染逻辑，确保主线程 UI 操作流畅。

### 4.4 增强元数据 (Article Metadata)
- **统计**：自动计算“阅读耗时（以 400 字/分钟计算）”与“文章总字数”。
- **版权**：支持 `CC BY-NC-SA 4.0` 协议展示。
- **更新看板**：在文章顶部显示“最后修改日期”，而非仅显示发布日期。

---

## 5. SEO 与 社交分享
- **JSON-LD**：自动生成 Schema.org 的 `Article` 结构化数据。
- **Open Graph**：为每一篇文章生成动态封面图或引用文章首图。
- **RSS**：提供全量或摘要式的订阅支持。
