# Aura 8 级模型路由：多维度评分下的性价比最优解

![Aura 模型路由可视化](featured.png)

让一个昂贵的旗舰模型去执行简单的文件归档任务是极大的资源浪费。Aura 通过内置的 **8 级模型路由（Model Router）**，为每一个 Matrix 节点实现了精细化的算力调度。

## 1. 8 级性能阶梯

我们将所有可用的模型（包括本地与远程）划分为 8 个等级，对应 3D 寻址中的 **Model 维度**。
- **L1-L3**：本地超轻量模型，负责简单正则解析、文本分类等高频、低延迟任务。
- **L4-L6**：中量级远程模型，负责常规的代码逻辑生成与长文总结。
- **L7-L8**：旗舰级推理模型，负责复杂的架构设计、意图分解与冲突解决。

## 2. 四维度动态评分

路由决策基于对每个模型的实时四维度评估：
- **性能（Performance）**：模型在当前任务类型上的历史成功率。
- **延迟（Latency）**：P99 级别的响应时间表现。
- **成本（Cost）**：Token 单价与预期消耗。
- **安全性（Safety）**：模型产生幻觉的统计概率。

## 3. 自动降级与重路由

如果高等级模型出现 API 故障或响应超时，路由引擎会自动触发**优雅降级**。Meta 会动态修改节点指针的 Model 维度，将任务自动重定向到备用的本地模型或其它供应商模型，确保整个工作流的永不断连。
